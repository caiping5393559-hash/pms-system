# TikTok 标签打印脚本版本记录

## v11.51 - 2026-06-29

- 快捷打印当天订单和复核扫描撤掉高风险 fast 大步滚动采集，改为小步安全采集。
- 每页采集数量可疑时自动补扫最多 3 轮。
- 当页面是 100/Page 且还有下一页，但采集数量不足时，快捷打印不生成打印任务，复核显示“中止/漏扫风险”，避免少单被当成完整结果。
## v11.50 - 2026-06-29

- 快捷打印当日订单启用快速扫描模式：减少每页固定等待、加大滚动步长、减少翻页后的固定等待。
- 只影响“快捷打印当日订单”；复核扫描仍使用原来的稳模式，避免漏扫。
- 状态栏显示每页扫描耗时和总耗时，方便后续继续优化。

## v11.49 - 2026-06-28

- 批量打印安全升级：当前页批量、快捷打印、复核批量只要发现本地或云端已打印订单，就停止整批打印。
- 单独补打保留：从历史里的单个订单点补打时，会先确认订单号、后六位、Title、Note、本地打印次数、云端打印次数。
- 单独补打发现云端 Title 和当前 Title 不一致时直接停止，避免把旧标签补到错误商品。

## v11.48 - 2026-06-28

- 新增打印前安全闸：同批重复订单号直接拦截，避免一个订单在同一批任务里出多张标签。
- Title 为空、未识别、Seller note 字段错位、订单号字段错位时停止打印。
- 同一订单历史 Title 与当前 Title 明显不一致时停止打印。
- 当前页批量、快捷打印、复核批量打印会先查云端打印次数；云端或本地已打印时必须二次确认。

## v11.47 - 2026-06-28

- 收窄标签订单号后 6 位数字。
- 增加订单号右侧安全边距，避免最后一位打印出格。

## v11.46 - 2026-06-27

- 调整 50x30 标签右侧三段文字布局。
- 订单号、Title、日期时间之间增加段距，避免三段文字挤在一起。

## v11.45 - 2026-06-27

- 修复本地 Chrome 扩展报错：打印窗口内联脚本违反 CSP，被 Chrome 拦截。
- 打印窗口不再写入 `<script>`，发送打印任务和关闭按钮改由外层插件绑定。
- 保留 v11.44 的可见打印窗口逻辑，避免二维码图片未加载时直接发任务。

## v11.42 - 2026-06-27

- 新增主菜单按钮“快捷打印今日拍卖”。
- 自动翻页扫描当天订单，遇到早于今天的订单后停止。
- 打印条件：今天 + `LIVE Auction`/`LIVE Giveaway` + 非 `Canceled` + 历史打印次数 `0` + 有可打印 Note。
- Giveaway 标签 Note 改为 `Giveaway` 加原 Seller note；原 note 为空时打印 `Giveaway`。
- 扫描结束后统一按 Title 排序批量打印，方便同款连续出标。

## v11.41 - 2026-06-27

- 串改猴 `@match` 从 `https://*/*` 收窄到 TikTok Seller 页面，避免其它网站也加载整套脚本。
- 云同步改为按需启动：没有待同步队列时，打开 TikTok 页面不再主动加载云 SDK/补写队列。
- 保留 v11.40 的 50x30 标签新版布局。

## v11.40 - 2026-06-27

- 右侧订单号、Title、日期时间整体下移，字体放大并压缩行距。
- Note 区域下移并放大，减少上方三段文字和备注之间的空隙。
