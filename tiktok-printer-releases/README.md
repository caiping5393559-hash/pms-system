# TikTok Printer Releases

This repository stores release files for the TikTok Seller label printer userscript.

- `latest.user.js`: latest install/update file
- `manifest.json`: latest version metadata and history
- `CHANGELOG.md`: release notes
- `tiktok-label-printer-v*.user.js`: versioned snapshots

After pushing to GitHub, use the raw `latest.user.js` URL as the stable download source for other computers.
